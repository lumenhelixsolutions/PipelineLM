Troubleshooting

Common Issues

"Manifest file is missing or unreadable"
Make sure you select the folder containing manifest.json, not the .zip file.

"Could not load icon"
Ensure the icons/ folder is inside the pipelinelm-pro folder alongside manifest.json.

"Service worker registration failed"
Check that background/sw.js exists. Try reloading the extension from the extensions page.

Side panel doesn't open
Click the ⚡ icon in your browser toolbar. If it's not there, pin it from the extensions menu.
